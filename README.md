<h1 align="center">Ghanapolitan</h1>

**A modern, full-stack Next.js news app built with ❤️ and TypeScript!**
Bootstrapped with create-next-app, this project is ready for development and deployment 🚀. It currently has a solid foundations structure with organized folders for ads, components, data, hooks, API utilities, styles, and more.

---

## 📦 Features

* Next.js + React
* TypeScript for type safety
* Component-based architecture
* Clean folder structure (`app`, `components`, `lib/api`, `hooks`, etc.)
* Ready for scalable development 🛠️

## 🚀 Getting Started

These instructions will get your project up and running on your local machine.

### 🧰 Requirements

Before you begin, make sure you have installed:

✔ Node.js (v18+ recommended)
✔ npm or yarn

---

## 📥 Installation

1. **Clone the repo:**

   ```bash
   git clone https://github.com/latifiss/ghanapolitan.git
   cd ghanapolitan
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn
   ```

---

## 🔧 Development

### 🏃 Run locally

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open your browser and go to:
👉 `http://localhost:3000`

Any changes you make will auto-refresh 🎉

## 📁 Folder Structure

Here’s how the project is organized:

```
├── ads/  
├── app/  
├── components/  
├── data/  
├── hooks/  
├── lib/api/  
├── public/  
├── store/  
├── styles/  
├── types/  
├── utils/  
├── package.json  
├── next.config.ts  
└── tsconfig.json
```

Each directory serves a purpose:

🧱 `app/` – Main application routes & pages
📦 `components/` – Reusable UI components
📊 `data/` – Static data files
⚙ `lib/api/` – API helpers
📐 `store/` – State management
🎨 `styles/` – Global and utility styling
🔌 `hooks/` – Custom React hooks
